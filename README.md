
# Roll The Dice!

Roll The Dice! is both a Solo & turn-based Multiplayer game in which the players must use aleatory events & probabilities to their very own advantage.


## Context

This project has been developed in the context of a competence assessment at Studi.
## How to Play (Rules)

### General rules

In order to win, a player needs to accumulate 100 points before his opponent. 
In each round, the designed player must roll a dice, and whatever number it gives, is the amount of
points that the player will start accumulating for the round.
From there, the player has two options:

- Take the points home, and let their opponent attempt to accumulate points
- Roll the dice again, and try to accumulate more points, but if the dice lands on 1, they will lose all their progress for the round

#### Note: 

There is no limit to how many times a player can roll the dice before the round ends.

### Solo mode
The player starts by rolling the dice. They must compete against a bot who got a pretty good risk tolerance...

### Multiplayer mode (2 Players max)

Each player starts by choosing their username.

Whoever starts the first round is chosen randomly, so no need to fight!


## Features

- Responsive & Mobile Friendly
- Solo Mode
- Multiplayer mode (Turn-based)
- Live Score Counters
